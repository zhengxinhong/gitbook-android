# TextView

### 1.基本属性

| 属性 | 关键字 | | 
|--| -- | -- |
| 宽度设置 | layout_width | |
| 高度设置 | layout_height | |
| 背景颜色 | background | |
| 文本内容 | TextView | |
| 字体颜色 | textColor | |

#### 宽度设置
android:layout_width="wrap_content"

#### 高度设置
android:layout_height="wrap_content"

#### 背景颜色
android:background="#000000"

#### 设置显示的文本内容
android:text="TextView文本内容"

#### 设置字体颜色
android:textColor="#EA5246"
